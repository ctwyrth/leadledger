# LeadLedger
LeadLedger is a lite-weight CRM/Project Tracker for freelancers and small service businesses.

## Purpose
This project demonstrates a full-stack MERN application focused on practical business workflows:
- client intake
- lead tracking
- project status management
- follow-up

## Tech Stack
- MongoDB
- Express
- React
- Node.js
- Vite

## Planned Features
- Client records
- Lead/opportunity tracking
- Project status pipeline
- Notes and activity history
- Basic dashboard metrics
- User authentication
